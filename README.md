### Example

```dart
Widget buildDTH(String text, String highlightText) {
  return DynamicTextHighlighting(
    text: text,
    highlightText: highlightText,
    highlightColor: Colors.yellow,
    style: TextStyle(
      fontSize: 18.0,
      fontWeight: FontWeight.bold,
    ),
  );
}
```
